# 콘텐츠 디렉터리

이 폴더에는 영가회 아카이브의 본문이 들어갑니다.

## 폴더 구조

```
content/
└── articles/
    ├── yeon-gi/      ← 一. 연기(緣起)
    ├── moim/         ← 二. 모임
    ├── geul/         ← 三. 글
    ├── saram/        ← 四. 사람
    └── jachwi/       ← 五. 자취
```

## 글 한 편 추가하기

각 장 폴더 아래에 `*.md` 파일을 한 개 만들면 자동으로 해당 장의
글 목록과 본문 페이지에 나타납니다. 파일 이름이 곧 URL slug가 됩니다.

### 프런트매터(머리글) 형식

```markdown
---
title: 제목 (필수)
subtitle: 부제 (선택)
date: 2026-04-01     # YYYY-MM-DD, 정렬에 사용 (필수)
author: 글쓴이      # 선택
excerpt: 한두 줄 미리보기 (선택)
---

본문 마크다운…
```

장 자체(이름·번호·설명)는 [`lib/chapters.ts`](../lib/chapters.ts)에서
수정합니다.
